Function getUserLikedPosts

Retrieves all posts liked by a specific user.

This function queries the post_likes table to find all posts that the specified user has liked, and then retrieves the corresponding posts from the posts table. It returns a list of posts that the user has interacted with by liking them.

Throws

Throws an error if retrieving the liked posts fails.

Example

// Get all posts liked by the user with ID 2
getUserLikedPosts(2)
  .then(likedPosts => console.log('Liked posts:', likedPosts))
  .catch(error => console.error('Error getting liked posts:', error));

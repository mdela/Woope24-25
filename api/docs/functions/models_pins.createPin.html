Function createPin

Creates a new pin with location data and optional metadata.

This function inserts a new record into the pins table with the specified pin ID, user ID, geographical coordinates (longitude and latitude), and optional metadata. The inserted pin record is returned upon successful creation.

Throws

Throws an error if the pin creation fails.

Example

// Create a new pin for user ID 1 at specific coordinates with optional metadata
createPin(101, 1, -73.935242, 40.73061, "Favorite spot")
  .then(pin => console.log('Pin created:', pin))
  .catch(error => console.error('Error creating pin:', error));
